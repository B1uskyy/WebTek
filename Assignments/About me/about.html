Questions

  1. The server manage most resources and services that the cliient needs and consumes. The server also hosts and delievers there resources.
  2. The client is what requests a service, such as files, data or internet access from the server and connects with the servers via a network.
  3. An IP address stands for "internet protocol" and is a unique address for a network or device. It works as an identifier.
  4. DNS is what translates human domain names to IP adresses so that they can be found and sent to a user. Every website has a unique IP address and the DNS makes it so we as humans dont need to remember these addresses, but just the domain name.
  5. The World Wide Web (WWW) is a part of the internet, we use the WWW for information exchage. And this information is facilitated throught the internet. So the internet is the connection, and the Web is the medium used.
  6. HTML og Hyper Text Markup Language is a language that is used for web page creating. It is used for structure of a page, such as paragraphs, sections, headings and more. HTML is not considered a programming language as nothing dynamic can be made with it.
